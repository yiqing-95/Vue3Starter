## api 调用是Promise

Promise的调用 js中有两种方法 

- 使用 then 
    这种方案 一般可以自己处理 错误情况 

- 使用 async/await 方案
    这种方案 一般只处理成功情况 错误情况在axios中做了统一的拦截处理 目前是主流做法 比较省事 统一 
    直接拿请求成功的结果

⚠️注意 平常的方法 加async修饰也是没问题的 就是里面可以不出现await 对promise的调用
